This network visualizes the relationships between hip hop artists from Atlanta and their producers that are found in this Wikipedia list. The wider edges represent more collaborations between a particular artist and a producer. The nodes that are larger indicate that the producer worked with more of the artists

The visualization represents 9,009 songs that were collected from the Genius API using this Jupyter Notebook. It is important to note that only 111 out of 162 (68%) of the artists listed at Wikipedia were found in Genius. If you would like to visualize the data you can download the Cytograph session file, the edge list, or the song data that was used.
